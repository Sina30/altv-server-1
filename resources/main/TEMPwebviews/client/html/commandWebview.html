Listes des Commandes

Touches

F1 : NoClip

F2 : Ouvrir ce menu

& : Allumer/Eteindre moteur d'un véhicule

* : Ragdoll

B : Pointer du doigt


Player

/pm : Permet de changer de Playermodel

/clothes : Permet d'ouvrir le menu de modification de l'apparence du playermodel (uniquement les modèles de base Femme et Homme)

/saveclothes [number] : Permet de sauvegarder un preset de vêtement

/setclothes [number] : Permet de récupérer un preset de vêtement sauvegardé

/goto [location]: Permet de se téléporter à une destination prédéfinie

/tp [id] or [playername]: Permet de se téléporter à un joueur

/tpm : Permet de se téléporter au point GPS. Marche également dans un véhicule

/coords [x] [y] [z] : Permet de se téléporter à une certaine coordonnée

/getpos : Permet d'obtenir les coordonnées actuelles


Véhicules

/veh : Permet d'ouvrir le menu de spawn des véhicules

/mod : Permet de modifier le véhicule actuel

/repair : Permet de réparer le véhicule actuel ou le plus proche si pas dans un véhicule

/destroy : Permet de detruire le véhicule actuel

/enter : Permet de forcer l'entrée dans un véhicule
